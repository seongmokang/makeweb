INTRO

강사 소개

현직 대기업 롯데 · SK · 현대 계열의 웹/앱 서비스 기획 경험을 바탕으로, 실제 서비스가 어떻게 만들어지고 운영되는지 알려드립니다.

이 강의는 단순히 코드를 가르치는 수업이 아니라, 원하는 웹사이트를 스스로 만들고 운영할 수 있는 실행력을 만드는 데 초점을 둡니다.

PURPOSE

강의 목적

내가 원하는 웹사이트를 만들기

아이디어를 기능 단위로 나누고, AI를 활용해 실제로 동작하는 사이트를 직접 만듭니다.

비개발자도 가능한 제작 방식 익히기

복잡한 디자인 툴과 깊은 프로그래밍 이론보다, 지금 당장 만들 수 있는 흐름에 집중합니다.

운영 가능한 서비스까지 연결하기

데이터 저장, AI 기능 연결, 배포, 이후 개선 방향까지 한 번에 이해할 수 있도록 돕습니다.

각 회차마다 눈에 보이는 결과물을 남기는 구조로 설계했습니다.

1회차

목표 정의 + 서비스 구조 설계

목표: 내 서비스 뼈대 만들기

  • 바이브 코딩 개념 이해
  • 만들 서비스 선정
  • 핵심 기능 3개로 기능 쪼개기
결과물: 기능 리스트 + 구조
2회차

개발 환경 + AI 사용법

목표: 개발환경 구축과 AI 사용법 이해

  • Claude Code로 코드 생성
  • Cursor 세팅
  • 기본 프로젝트 생성
결과물: 프로젝트 실행 화면
3회차

첫 화면 만들기

핵심: 눈에 보이는 결과 만들기

  • 페이지 생성
  • 버튼 / 입력창 구성
  • 기본 UI 만들기
결과물: 메인 페이지 완성
4회차

핵심 기능 1 구현

핵심 기능을 실제로 작동시키는 시간

  • 사용자 입력 → 결과 출력
  • 상태 관리 개념 이해
결과물: 작동하는 기능 1개
5회차

데이터 저장 (백엔드 최소화)

서비스답게 데이터를 다루는 단계

  • 데이터 저장 / 불러오기
결과물: 사용자 데이터 저장 가능
6회차

핵심 기능 2 + AI 연결

AI를 붙여 서비스의 차별화 만들기

  • AI 기능 연결
  • 추천 / 요약 / 자동화 실습
결과물: AI가 들어간 서비스
7회차

배포

내 서비스가 실제 인터넷에 공개되는 순간

  • Vercel 배포
  • 실제 URL 생성
결과물: 내 서비스 인터넷 공개
8회차

개선 + 수익화 구조

완성 후의 운영과 성장 방향 설계

  • 사용자 흐름 개선
  • 수익 모델 붙이기
결과물: 서비스 완성 + 방향성

DIFFERENCE

왜 바이브 코딩 강의인가

일반적인 웹사이트 제작 방식

  1. 설계 — Figma를 활용한 와이어프레임 설계
  2. 디자인 — 컨셉과 목적에 맞춘 디자인 구조 구축
  3. 개발 — 개발 플랫폼과 언어 선택 후 구현
  4. 운영 — 보안, 서비스 관리, 유지보수

바이브 코딩 방식

설계와 디자인을 길게 준비하는 대신, 바로 개발을 시작해 빠르게 서비스 형태를 만듭니다.

  • 아이디어를 빠르게 화면과 기능으로 검증
  • AI를 활용해 설계 · 구현 · 수정 속도 향상
  • 운영 단계의 보안과 서비스 관리까지 AI로 보조 가능
